# ⚔️ **StrengthSMP | Professional Strength PvP System**

**StrengthSMP** is the ultimate competitive combat overhaul. Now **Ultra-Lightweight** and **Universally Compatible** across all modern Minecraft versions.

### 🚀 **What's New in v1.1.0 (The Weapon Soul-Binding Update)?**
* **Weapon Soul-Binding (`/bind`):** Bind your weapon class to your soul. Keep your bound weapon on death, and watch it spin/fly back to your hand upon respawning via a custom Loyalty return flight animation!
* **Cinematic Crate Shuffle GUI:** Right-clicking Reroll Books now triggers an animated scrolling interface (`WeaponShuffleGUI`) with dynamic deceleration, thunder sounds, and lightning/totem particles!
* **Advanced Statistics Grimoires:** Tracks Souls Harvested (with dynamic lore listing recent PvP victims), Damage Dealt, Soul Sacrifices, Duration Bound, and Active Ability Triggers inside the `/bind` GUI.
* **PlaceholderAPI Support:** Expanded placeholders including `%strengthsmp_has_bound_weapon%` and `%strengthsmp_bound_weapon_class%`.
* **Universal Compatibility (1.16.5 - 1.21.1+):** Smooth reflection-based skin loaders and dynamic retro-compatible sound/particle resolvers to prevent legacy server crashes.
* **Optimized Size:** Shaded jar optimized down to a super-lightweight **138KB**.

## ⌨️ Documentation & Commands

### Player Commands
* `/bind` - Open the Soul-Binding menu (requires 3 strength to bind, 1 to swap).
* `/skills` - Open the Player Skill tree.
* `/strength` - View your current stats.
* `/withdraw <amount>` - Turn strength into Nautilus Shell items.
* `/ability` - Activate weapon ability.

### Admin Commands (strengthsmp.admin)
* `/strengthsmp reload` - Reload configurations and Discord webhooks.
* `/strengthsmp setstrength <player> <amount>` - Manually set player strength.
* `/strengthsmp setweapon <player> <weapon>` - Set a player's class weapon.
* `/strengthsmp reroll <player>` - Reroll a player's class via the Weapon Shuffle GUI.
* `/punish <player>` - Force double strength loss.
* `/killreset` - Reset global seasonal leaderboards.
* `/notimer <player>` - Toggle safety protection.
